9. Linux Privilege Escalation

9.1. System Enumeration 
a. uname -a #Give details about system and OS
b. cat /proc/version #Give OS version details
c. lscpu #Give memory details
d. ps aux #Give running services details

9.2. User Enumeration 
a. Whoami #Show who is the current user
b. id #Show user name and id
c. sudo -l #show file that do not need root users
d. cat /etc/passwd #show username without password
e. cat /etc/passwd | cut -d : -f 1 #Grepping only username from passwd file
f. cat /etc/group #show group names
g. history #shows all previous commands

9.3. Network Enumeration 
a. ifconfig #show Ip addresses
b. ip a #Shows IP address
c. ip route #Network pathways
d. arp -a #Shows ARP table
e. ip neigh #Shows ip state
f. netstat -ano #Shows ports and traffic flow

9.4. Password Hunting
a. grep --color=auto -rnw '/' -ie "PASSWORD=" --color=always 2> /dev/null #shows all the password word filtering from  all logs 
b. locate password | more #locate password files and directories
c. find / -name id_rsa 2> /dev/null #finding id rsa keys
d. find / -name authorized_keys 2> /dev/null #finding authorized keys
